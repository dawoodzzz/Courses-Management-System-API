# Courses Management API

A simple RESTful API to manage courses using Node.js, Express, and MongoDB.

## Features
- Get all courses.
- Get a single course by ID.
- Add a new course with validation.
- Update course data.
- Delete a course.

## Tech Stack
- Node.js
- Express.js
- MongoDB & Mongoose
- Express-Validator

## How to Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Set your MongoDB URL in the code or `.env` file.
3. Start the server:
   ```bash
   npm run run:dev
   ```
